# `@starwind-ui/astro`

Generated Astro Primitive adapters for Starwind UI Runtime. The package is source-published and
supports Astro 5 or newer.

```bash
npm install @starwind-ui/astro@beta
```

Import component families from subpaths, for example `@starwind-ui/astro/select`. Add
`ThemeInitScript` from `@starwind-ui/astro/theme` in the document head to apply the stored theme
before paint. Runtime-backed adapters clean up their controllers across Astro client navigation.

Color Picker remains a legacy Astro migration surface and is not exported by this package.
