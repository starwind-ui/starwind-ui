# `@starwind-ui/react`

Generated React Primitive adapters for Starwind UI Runtime. The package supports React and React
DOM 18 or newer.

```bash
npm install @starwind-ui/react@beta
```

Import component families from subpaths, for example `@starwind-ui/react/select`. The `starwind`
beta CLI configures current Vite React projects with Tailwind, the `@/*` alias, the Starwind CSS
entry, and Theme prepaint initialization.

Color Picker is not available for React in this beta.
